```

    |--Ann_Veneman_0006.jpg
    |--Ann_Veneman_0007.jpg
    |--...
|--Catherine_Zeta-Jones
    |--Catherine_Zeta-Jones_0006.jpg
    |--Catherine_Zeta-Jones_0007.jpg
    |--...
|--...
```
